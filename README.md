# sales-analysis-automation 🚀
## Overview
This project began as a simple sales data automation script and gradually evolved into a complete analytics dashboard.
Users can upload Excel sheets datasets and instantly generate:
- Interactive chart
- KPI Summeries
- Revenue Trends
- Automated PDF Reports
  
# Supported Excel Format 📁
The current version of the this project expects Excel files containing the following coloumns:
- Sales Data
- Product Name
- Units Sold
- Revenue
- Geographic Region
Example Files (Sales_1.xlxs...) are included in this repository

## Project Evolution 📊
This project was created in different phases, with  each phase adding new functionality.

### Phase 1: Fictional Dataset Generation
- Generated three fictional excel sales datasets using Powershell
- Created realistic sales records for testing and demonestration

### Phase 2: Automated Fictional Reporting 
- Compared all three Excel sales datasets
- Generated a professional PDF summerized report of revenue trends and insights

### Phase 3: Interactive Web Dashboard
To take this project one step ahead,
- Built a streamlit Dashboard
- Added KPIs, revenue trends, and regional analysis

### Phase 4: User upload Support
- added support for uploading excel files but with required format
  

## Features ✨
- Upload multiple Excel files
- Interactive Dashboard
- Identify sales trends
- Create professional charts
- Generate a PDF financial report automatically
- KPI Cards
- Dark theme UI
